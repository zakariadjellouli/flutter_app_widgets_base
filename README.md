# Atelier Flutter 02 - Widgets de Base

Application Flutter créée pour découvrir les widgets de base en développant une interface statique et esthétique.

## Étapes Réalisées
- Étape 1 : Création du projet Flutter
- Étape 2 : Ajout d'une Column avec deux textes
- Étape 3 : Implémentation d'un Container stylisé
- Étape 4 : Découverte du Stack avec superposition
- Étape 5 : Ajout d'une Row avec des icônes
- Étape 6 : Intégration d'une deuxième Column
- Étape 7 : Personnalisation de l'interface
