- Étape 1 : Créer un projet Flutter

- ##   Widgets Implémentés
- **Column** - Disposition verticale principale
- **Row** - Barre d'icônes horizontale
- **Stack** - Superposition texte/icône
- **Container** - Boîte stylisée orange
- **Text** - Contenu textuel personnalisé
- **Icon** - Icônes Material Design
